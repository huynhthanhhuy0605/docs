## Troubleshooting

<AccordionGroup>
<Accordion title="I didn't receive the code">
Make sure you:
- Downloaded the **mobile app** (iOS or Android) - this doesn't work on the web version
- Created a note with the exact key **"student"** (lowercase) in the content field
- Logged in to your NoteMe account

Try refreshing the app or creating another note with "student" key if needed.
</Accordion>

<Accordion title="Can I use this on the web version?">
No, this step is only available on the mobile app (iOS or Android). You need to download the app from the App Store or Google Play.
</Accordion>

<Accordion title="Do I need a Pro NoteMe subscription?">
No! This is available to all NoteMe users, including free users. You just need to download the mobile app and create a note with the key "student".
</Accordion>
</AccordionGroup>
