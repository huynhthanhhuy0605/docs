## General questions

<AccordionGroup>
<Accordion title="What is NoteMe?">
NoteMe is a note-taking and reminder system that helps you capture everything you need to remember in one place. Create notes from mobile app, Canva, or automation workflows, and access them all from your mobile app.
</Accordion>

<Accordion title="Is NoteMe free?">
Yes, NoteMe offers a free plan with basic features and a limited number of notes. Pro plans unlock unlimited notes, Canva App access, and advanced features. Visit [noteme.qrdyai.com](https://noteme.qrdyai.com) to see current pricing for your platform.
</Accordion>

<Accordion title="Do I need design skills to use NoteMe?">
No, NoteMe is designed for everyone. It's a simple note-taking and reminder app that anyone can use. No design or technical skills needed.
</Accordion>

<Accordion title="Who is NoteMe for?">
NoteMe is perfect for:
- General users who want to capture notes and reminders
- Designers working in Canva who need to save project notes
- People who want to automate note creation from various sources
- Anyone who needs a central place to remember things
</Accordion>
</AccordionGroup>

- Android (Google Play)

<Accordion title="How do I create notes with AI?">
When creating a note, you can use AI assistance to help format and organize your thoughts. Simply describe what you want to remember and AI will help create a well-structured note.

<Accordion title="How do tags work?">
Tags help you organize notes. Add one or more tags to each note, then search by tag to find all notes with that tag. Create a consistent tagging system for best results.
</Accordion>

<Accordion title="Can I set reminders?">
Yes! When creating or editing a note, you can set a date and time reminder. NoteMe will notify you at the specified time. View all reminders in Calendar view.
</Accordion>

<Accordion title="How do I search my notes?">
Use the search function to find notes by keyword or tag. You can search through note content, titles, and tags to quickly locate what you're looking for.
</Accordion>

## Canva app

<AccordionGroup>
<Accordion title="How do I install NoteMe in Canva?">
1. Open Canva and click "Apps" in the left sidebar
2. Search for "NoteMe"
3. Click "Add" or "Use" to install the app
</Accordion>

<Accordion title="Do I need Pro to use NoteMe in Canva?">
Yes, the Canva App requires a Pro subscription. Free users can use the mobile app and web version at [noteme.qrdyai.com](https://noteme.qrdyai.com).
</Accordion>

<Accordion title="What can I do with NoteMe in Canva?">
You can create notes directly while designing, save project ideas, maintain checklists, store reference links, and set project reminders - all without leaving Canva.
</Accordion>

<Accordion title="Do notes created in Canva sync to mobile?">
Yes! All notes created in the Canva App automatically sync to your NoteMe mobile app. You can access them from anywhere.
</Accordion>
</AccordionGroup>

</AccordionGroup>

## Troubleshooting

<AccordionGroup>
<Accordion title="Notes not syncing between platforms">
Make sure you're logged into the same NoteMe account on all platforms. Check your internet connection and try refreshing the app or logging out and back in.
</Accordion>

<Accordion title="Can't access Canva App">
Ensure you have a Pro subscription active. The Canva App is only available for Pro users. Free users can use the mobile app and web version.
</Accordion>

<Accordion title="Reminders not working">
Check your device notification settings to ensure NoteMe has permission to send notifications. Also verify that the reminder date and time are set correctly.
</Accordion>

<Accordion title="Can't find a note">
Use the search function to search by keyword or tag. Make sure you're using the correct tags if searching by tag. Check if you're looking in the right account if you have multiple accounts.
</Accordion>
</AccordionGroup>
