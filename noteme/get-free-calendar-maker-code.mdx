## Troubleshooting

<AccordionGroup>
<Accordion title="I didn't receive the code">
Make sure you've completed all steps correctly:
- Downloaded the **mobile app** (iOS or Android) - this doesn't work on the web version
- Created a note with the exact key **"calendar"** (lowercase) in the content field
- Logged in to your NoteMe account

If you've completed all steps correctly and still don't see the code, try:
- Refreshing the app or restarting it
- Checking if the note was saved successfully
- Creating another note with "calendar" key

If the issue persists, contact support at [support@qrdynamic.xyz](mailto:support@qrdynamic.xyz).
</Accordion>

<Accordion title="The code doesn't work">
Make sure you:
- Selected the **1-month subscription plan** (not annual or other plans)
- Entered the code exactly as shown (check for any extra spaces or characters)
- The code hasn't expired (check the notification for expiration date if mentioned)

If the code still doesn't work, contact support with a screenshot of the error.
</Accordion>

<Accordion title="Can I use this on the web version?">
No, this offer is only available on the mobile app (iOS or Android). You need to download the app from the App Store or Google Play to participate.
</Accordion>

<Accordion title="Can I use this code if I already have Calendar Maker?">
This offer is typically for new users. If you already have an active Calendar Maker subscription, the code may not apply. Contact support for clarification.
</Accordion>

<Accordion title="Do I need a Pro NoteMe subscription?">
No! This offer is available to all NoteMe users, including free users. You just need to download the mobile app and create a note with the key "calendar".
</Accordion>

<Accordion title="What if I typed 'Calendar' with capital C?">
Make sure to use lowercase "calendar" as the key. The system looks for the exact key "calendar" (all lowercase) in your note content.
</Accordion>
</AccordionGroup>
