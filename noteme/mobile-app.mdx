## Creating notes

<Steps>
<Step title="Open the app">
Launch NoteMe on your mobile device or visit [noteme.qrdyai.com](https://noteme.qrdyai.com) on web.
</Step>

<Step title="Tap create">
Tap the "+" button or "Create Note" button to start a new note.
</Step>

<Step title="Use AI assistance">
Let AI help you create notes faster. Describe what you want to remember and AI will help format it.
</Step>

<Step title="Add details">
Add text, links, tags, or set a reminder date and time for your note.
</Step>

<Step title="Save">
Tap save and your note is ready. It will sync across all your devices automatically.
</Step>
</Steps>

## Setting reminders

<Steps>
<Step title="Create a note">
Start by creating a new note or editing an existing one.
</Step>

<Step title="Set reminder">
Tap the reminder option and select a date and time when you want to be notified.
</Step>

<Step title="Choose notification">
Choose how you want to be reminded - notification, email, or both.
</Step>

<Step title="Save reminder">
Save your note with the reminder. NoteMe will notify you at the specified time.
</Step>
</Steps>

## Organizing with tags

<CardGroup cols={2}>
<Card title="Add tags" icon="tags">
Add one or more tags to each note. Tags help you organize and categorize your notes.
</Card>

<Card title="Search by tags" icon="magnifying-glass">
Use the search function to find all notes with a specific tag. Quickly filter your notes.
</Card>

<Card title="Multiple tags" icon="layer-group">

</CardGroup>

## Saving links

<Steps>
<Step title="Copy link">
Copy any link you want to save from your browser or other apps.
</Step>

<Step title="Create note with link">
Create a new note and paste the link. Add a description or context if needed.
</Step>

<Step title="Add tags">
Tag the link note for easy retrieval later. For example, tag as "reference", "tutorial", or "inspiration".
</Step>

<Step title="Access anytime">
All your saved links are accessible in NoteMe. Search by tag or keyword to find them quickly.
</Step>
</Steps>

## Calendar view

<Steps>
<Step title="Open calendar">
Tap the Calendar icon to view all your reminders in calendar format.
</Step>

<Step title="Browse by date">
Navigate through days, weeks, or months to see upcoming reminders.
</Step>

<Step title="View details">
Tap on any reminder in the calendar to see full note details and make changes if needed.

</Steps>
